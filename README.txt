==========================================================
  BUSCADOR DE RESULTADOS DE ATLETISMO - CADA
==========================================================

QUE ES
------
Un buscador de 121.157 resultados de atletas de competencias de la
Confederacion Argentina de Atletismo (CADA).
Anios incluidos: 2007-2014 y 2022-2026.

COMO USARLO
-----------
1. Hace doble-clic en "Buscador Atletismo CADA.html".
   Se abre en tu navegador (Chrome, Edge o Firefox).
   NO necesita internet, ni Python, ni instalar nada.

2. Busca por: nombre y apellido, disciplina, localidad/provincia,
   anio, edad (rango) o categoria. Se filtra a medida que escribis.

3. Clic en el titulo de una columna ordena de menor a mayor;
   otro clic invierte (mayor a menor). La flecha ^/v indica el orden.

4. Boton verde "Imprimir / Guardar PDF": arma un PDF con TODOS los
   resultados que tengas filtrados en ese momento.

5. En cada fila, el enlace "abrir" abre el PDF original de esa
   competencia.

IMPORTANTE
----------
* Mantene el archivo HTML y la carpeta "resultados" SIEMPRE JUNTOS,
  en la misma carpeta. Si los moves de lugar, move los dos.
* Si abris el HTML y queda en "cargando base de datos...", proba con
  otro navegador (Chrome o Edge funcionan bien).
